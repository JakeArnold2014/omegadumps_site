Discord Bot Commands Reference

This document lists all available slash commands for Omegabot.

Portfolio Management Commands ▼

/portfolio add

Add an item to your portfolio.

Parameters:

  • item (string): Item name or ID
  • buy_price (integer): Buy price
  • quantity (integer): Quantity bought
  • target_price (integer, optional): Receive a DM if the latest price goes above this value (optional)
  • stop_loss (integer, optional): Receive a DM if the latest price goes below this value (optional)

/portfolio remove

Remove an item from your portfolio interactively.

Parameters:

  • Interactive selection - no parameters required

/portfolio view

View your current portfolio.

Parameters:

  • show_history (boolean, optional): Shows the individual buy price history for each item (optional)

/portfolio edit_alerts

Edit the target or stop loss for an item in your portfolio.

Parameters:

  • Interactive selection - no parameters required

/portfolio edit_history

Edit a buy history entry for an item in your portfolio.

Parameters:

  • Interactive selection - no parameters required

/portfolio trends

Show price trends for all items in your portfolio over the last 7 days.

Parameters:

  • No parameters required

/portfolio graphs

Generate price graphs for all items in your portfolio from their purchase dates.

Parameters:

  • No parameters required

Price Information Commands ▼

/price

Get current price and associated info for an item.

Parameters:

  • item (string): Item name

/trend

Show price change for an item over a period.

Parameters:

  • item (string): Item name or ID
  • period (string, optional): Time period (e.g. 7d, 24h, 1h, 30d, 6m, 12m) (default: 7d)

/trend-category

Show price change for a set of items (category) over a period.

Parameters:

  • category (choice): Item set to use
  • period (string, optional): Time period (e.g. 7d, 24h, 1h, 30d, 6m, 12m) (default: 7d)

/graph

Generate a price graph for an item with custom timeframe (up to 10 years).

Parameters:

  • item (string): Item Name
  • timeframe (string, optional): Timeframe (e.g., '30 minutes', '24 hours', '7 days', '1 month', '2 years', '10 years'). Default: 24 hours
  • theme (string, optional): Color theme for the graph (optional)

Recipe and Crafting Commands ▼

/recipe

Get current price and values for a specific recipe.

Parameters:

  • recipe (string): Recipe name
  • strategy (choice): Strategy to use for calculations

/recipe-find-profitable

Find all profitable recipes based on current market prices.

Parameters:

  • strategy (choice): Strategy to use for calculations

/recipe-favorites-add

Add a recipe to your favorites list.

Parameters:

  • recipe (string): Recipe name to add to favorites

/recipe-favorites-remove

Remove a recipe from your favorites list.

Parameters:

  • Interactive selection of recipe

/recipe-favorites-list

View your favorite recipes list.

/recipe-favorites-analyze

Analyze all your favorite recipes with current prices.

Parameters:

  • strategy (choice): Strategy to use for calculations

Misc Commands ▼

/notify

Notify you when an item's price goes above or below a threshold.

Parameters:

  • item (string): Item name or numeric ID of the item
  • price (integer): Price (in GP) at which you want to be alerted
  • direction (choice): Whether to alert when price is above or below this value
    • Above
    • Below
  • price_type (choice, optional): Which price to monitor
    • Either (both)
    • Instabuy (high)
    • Instasell (low)

/notifications

View and remove your current price alert notifications.

Parameters:

  • No parameters required

/deathscoffer

Find the best items to sacrifice to deaths coffer.

Parameters:

  • strategy (choice): Select the buy strategy
    • Insta Buy (high price)
    • Slow Buy (low price)
  • freshness_minutes (integer, optional): Maximum age of price data in minutes (default: 30 minutes)
  • minimum_difference (integer, optional): Minimum difference (gp) between sacrifice and buy price (default: 20,000)
  • min_price (integer, optional): Minimum GE price for an item to be considered (default: 10,000)
  • max_price (integer, optional): Maximum GE price for an item to be considered (default: 100,000,000)
  • min_margin_x_volume (integer, optional): Minimum (margin x 24h volume) for an item to be considered (default: 2,000,000)

/uptime

Displays the current uptime of the bot.

Parameters:

  • No parameters required

/checkstats

Check dump statistics for an item over a timeframe.

Parameters:

  • item (string): Item name or ID to check statistics for
  • timeframe (choice): Timeframe to analyze
    • 1 Hour
    • 6 Hours
    • 12 Hours
    • 24 Hours
    • 7 Days
    • All Time

/find-margins-beta

Find the top 10 margins using current market prices.

Parameters:

  • freshness_minutes (integer, optional): Maximum age of price data in minutes (default: 60, max: 1440)
  • min_margin (integer, optional): Minimum margin threshold in GP (default: 1000)
  • minimum_buy_price (integer, optional): Minimum buy value in GP (default: 1)
  • maximum_buy_price (integer, optional): Maximum buy value in GP (default: 2,147,483,647)
  • minimum_roi (float, optional): Minimum ROI percentage (default: 2.0)
  • sort_by_limit_profit (boolean, optional): Sort by limit profit (default: False)
  • min_margin_volume (integer, optional): Minimum margin × volume score (default: 500,000, higher = more liquid items)

Flipping Utility File Management Commands ▼

/edit_flipping_file

Select and edit an item entry from a flipping JSON file.

Parameters:

  • file (attachment): Upload your flipping JSON file found at <user>/.runelite/flipping

/combine_flipping_files

Upload multiple flipping JSON files and combine them into one.

/add_trade_entry

Add trade entries for an item to a flipping JSON file.

Parameters:

  • file (attachment): Upload your flipping JSON file
  • item_name (string): Name of the item to add trades for
  • quantity (integer): Quantity to add
  • price (integer): Price per item
  • status (string): Status (BOUGHT or SOLD)

/delete_trade_entries

Delete trade entries for an item from a flipping JSON file interactively.

Parameters:

  • file (attachment): Upload your flipping JSON file

Notes ▼

  • DM Only Commands: Some commands (like flipping utility commands) can only be used in direct messages with the bot, not in servers.
  • Autocomplete: Most commands support autocomplete for item names and other parameters.
  • Permissions: Some commands may require specific roles or permissions to use.
  • Hidden Commands: There are additional hidden commands available to administrators and bot owners that are not listed here as they are not intended for general use.

Usage Tips ▼

  1. Item Names: You can use either item names or numeric IDs for most commands that require an item parameter.
  2. Time Periods: For trend commands, use formats like "7d" (7 days), "24h" (24 hours), "1h" (1 hour), etc.
  3. File Uploads: For flipping utility commands, make sure to upload valid JSON files from your RuneLite flipping folder.
  4. Portfolio Management: Use the portfolio commands to track your investments and set price alerts.
  5. Recipe Analysis: The recipe commands help you find profitable crafting opportunities in the current market.

Support ▼

If you encounter any issues with these commands or need additional help, please contact the bot administrator or check the server's help channels.
